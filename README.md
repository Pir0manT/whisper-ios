# Whisper iOS

Оффлайн приложение для записи и транскрибации лекций на iPhone.

Использует [whisper.cpp](https://github.com/ggml-org/whisper.cpp) для локального распознавания речи — интернет не нужен.

## Возможности

- Запись аудио до 45 минут (стандартная лекция)
- Распознавание русской речи (модель `ggml-medium`, 1.5 ГБ)
- Режим реального времени
- Кнопка «Поделиться» — отправка текста в Telegram, Заметки и другие приложения
- Полностью оффлайн — работает без интернета

## Установка через AltStore (Windows)

Приложение не подписано Apple Developer сертификатом, поэтому устанавливается через AltStore.

### Предварительные требования

- iPhone с iOS 16+
- Windows PC
- USB-кабель Lightning/USB-C
- iTunes установлен на PC

### Шаги

1. **Скачайте AltServer** с [altstore.io](https://altstore.io)
2. **Установите AltServer** на Windows и запустите
3. **Подключите iPhone** к PC по USB, разблокируйте и доверьтесь компьютеру
4. **Установите AltStore** на iPhone через AltServer (иконка в трее → Install AltStore → ваш iPhone)
5. **Скачайте .ipa** из [GitHub Releases](https://github.com/Pir0manT/whisper-ios/releases)
6. **Откройте .ipa** в AltStore на iPhone (через «Открыть в AltStore» или через My Apps → +)

### Обновление подписи

AltStore подписывает приложения на 7 дней. Для автоматического обновления:
- Держите AltServer запущенным на PC
- iPhone и PC должны быть в одной WiFi-сети
- AltStore автоматически обновит подпись в фоне

## Использование

1. Запустите приложение
2. Разрешите доступ к микрофону
3. Нажмите **«Начать запись»** — кнопка станет красной
4. Нажмите **«Остановить»** для завершения записи
5. Нажмите **«Распознать»** — дождитесь обработки
6. Результат появится в текстовом поле
7. Нажмите **«Поделиться»** для отправки текста в Telegram или другое приложение

### Режим реального времени

Нажмите **«Реальное время»** перед записью — текст будет обновляться во время записи. Подходит для коротких записей, для длинных лекций лучше записать полностью и затем распознать.

## Ограничения

- Первый запуск: загрузка модели в память занимает несколько секунд
- Обработка 5 минут аудио занимает ~1-3 минуты (зависит от модели iPhone)
- Подпись через AltStore действует 7 дней (автообновление по WiFi)
- Размер приложения ~1.5 ГБ (модель встроена)

## Сборка

Приложение собирается автоматически через GitHub CI при создании тега:

```bash
git tag v0.1.0
git push origin main v0.1.0
```

CI на macOS runner:
1. Собирает whisper.xcframework из submodule whisper.cpp
2. Скачивает модель ggml-medium.bin с HuggingFace
3. Собирает Xcode проект без подписи
4. Упаковывает .ipa и загружает в GitHub Release
